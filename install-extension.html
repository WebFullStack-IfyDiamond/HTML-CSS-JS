Installation and use VS code's Live Server Extension

Install | Use | Video

Installation Instructions

  1. In visual studio code click the extension icon in the left column toolbar
  2. In the search bar that appears, type Live Server
  3. Select the Live Server extension. It should be the top choice.
  4. In the Live server page that appears, click the green install button.

Usage Instructions

  • Visual studio code must open to a folder to use Live Server
  • In VS Code's explorer, right click (windows) or control+click (macOs) on an HTML file. From the context menu that appears, select Open with Live Server.
